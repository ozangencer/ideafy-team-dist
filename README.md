# Ideafy Team — Distribution

Notarized DMG releases for **Ideafy Team**, the cloud edition of [Ideafy](https://ideafy.dev).

This repository **does not contain source code**. It exists only to host signed and notarized desktop binaries via GitHub Releases. The Solo (open-source) edition lives at [`ozangencer/ideafy`](https://github.com/ozangencer/ideafy).

Direct downloads (always latest):

- macOS Apple Silicon: [`Ideafy-Team-macOS-arm64.dmg`](https://github.com/ozangencer/ideafy-team-dist/releases/latest/download/Ideafy-Team-macOS-arm64.dmg)
- macOS Intel: [`Ideafy-Team-macOS-intel.dmg`](https://github.com/ozangencer/ideafy-team-dist/releases/latest/download/Ideafy-Team-macOS-intel.dmg)
